# ESPACE

ESPACE 是一個基於 Node.js + Express + EJS + MySQL + Sequelize 架構的資產預約與維護管理系統，採用分Schema隔離的多租戶架構，搭配 Docker 部署於 AWS Lightsail，以 RESTFul API 滿足網站不同資料的互動需求，搭配 Docker 部署，打造一個多功能的電商網站。
