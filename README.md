# ESPACE

ESPACE 是一個基於 Node.js + Express + EJS + MySQL + Sequelize 架構的資產預約與維護管理系統，採用分Schema隔離的多租戶架構，搭配 Docker 部署於 AWS Lightsail 的生產級應用，本頁面會展示應用介紹與部份程式碼。
